## Run Locally

Clone the project

```bash
  git clone https://github.com/YasinMahmoudi/bmi-calculator.git
```

Go to the project directory

```bash
  cd bmi-calculator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
